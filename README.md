# Proyecto de Turismo: Barcelona Responsive

## Descripción
Este proyecto consiste en una página web turística sobre la ciudad de Barcelona, diseñada para ser completamente **responsive** y accesible desde cualquier dispositivo (PC, Tablet y Móvil). 
La web incluye información sobre lugares emblemáticos, una galería de imágenes, ubicación interactiva y un servicio de reserva de tours virtuales.
